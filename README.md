🚀 數據科學與空間分析作品集 (Data Science & Geospatial Portfolio)
歡迎來到我的 GitHub 倉庫！這裡紀錄了我從基礎數據處理到複雜空間建模的學習歷程。本倉庫整合了多個專案，涵蓋了環境科學、都市安全、地理資訊系統（GIS）與氣候研究。

📂 核心專案導覽
1. 台北市 PM2.5 歷年時空規律分析 (2018-2023)
技術棧： Python, Pandas, GeoPandas, Matplotlib, Moving Average Algorithms

專案描述： 整合台北市 7 個測站、跨越 6 年的原始監測數據，重現空氣品質改善趨勢。

技術亮點：

演算法實現： 依照環境部規範，手動撰寫 12 小時與 4 小時加權移動平均邏輯計算 AQI 副指標。

視覺化強化： 使用非線性縮放技術 (AQI 
3.5
 ) 製作空間氣泡圖，顯著拉開年度間的視覺對比。

季節性分析： 透過盒鬚圖 (Boxplot) 深度剖析秋冬境外污染與夏季穩定空氣品質的差異。

2. 台北市犯罪地圖 (Interactive Crime Map)
技術棧： JavaScript, Leaflet.js / Folium, Spatial Clustering

專案描述： 開發網頁版互動地圖，將台北市刑事案件數據空間化。

技術亮點：

熱點分析： 透過空間聚類技術，識別都市犯罪熱區，提供視覺化的安全參考。

互動設計： 提供不同犯罪類型的篩選功能，並整合多圖層切換優化使用者體驗。

3. 台灣大學部分地區 3D 建模 (Campus 3D Modeling)
技術棧： LiDAR Data, Photogrammetry, SketchUp / Blender

專案描述： 對台灣大學特定區域進行高精度三維數位建模。

技術亮點：

精確測量： 結合實地影像與空拍數據，修復模型貼圖，還原校園建築細節。

空間應用： 此模型可作為校園導覽或空間規畫之數位孿生（Digital Twin）基礎。

4. 坪林區過去氣候分析 (Pinglin Climate Retrospective)
技術棧： Time-series Analysis, Seaborn, Climate Statistics

專案描述： 針對坪林地區長期的氣溫與降雨數據進行統計建模。

技術亮點：

氣候趨勢： 分析農業環境（茶園）受極端天氣影響的潛在風險。

數據清洗： 處理測站缺失值，並建立穩健的長期趨勢線。
